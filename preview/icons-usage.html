Usage rules
  • Stroke: 1.5 px · Style: line · Matches Figma /Icons/* 1:1 (709 icons).
  • Loaded via https://unpkg.com/lucide@latest; call lucide.createIcons() after mount.
  • Color always inherits via currentColor. Never set fill/stroke inline — use the parent's color.
  • Status icons (circle-check / alert-triangle / circle-x / info) MUST use semantic tones, not arbitrary blues.
  • Inside buttons: 14 px. In table rows: 16 px. Sidebar / nav: 20 px. Drawer headers: 24 px.
